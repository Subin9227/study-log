## 3. 데이터 시각화 (`0527_Visualization_Linda.ipynb`)

Matplotlib과 Seaborn을 이용한 다양한 시각화 기법을 학습했습니다. (4일 분량)

### 학습 내용

| 챕터 | 주요 개념 |
|---|---|
| 정형 데이터 | DataFrame 기본 정보 확인, 다중 조건 필터링 |
| 비정형 데이터 | JSON → DataFrame 변환, 문자열 정제, 문장 단어 수 분석 |
| 막대그래프 | 기본/누적/그룹형 막대 그래프 |
| 히스토그램 | 단일/다중 겹치기/누적 히스토그램 |
| 산점도 | 기본 산점도, 색상·투명도 설정, 그룹별 색상 구분 |
| 박스플롯 | 기본/다중/이상값 강조 박스플롯 |
| 고급 다중 그래프 | `subplots()`, X축 공유, `gridspec` 불규칙 레이아웃 |
| 벤 다이어그램 | 집합 연산, 조건부 색상 설정 |
| 범주형 데이터 | Seaborn `barplot`, `boxplot`, `violinplot` + `stripplot` |
| 연속형 데이터 | 히스토그램 + KDE, 사인 그래프, 회귀선 산점도 |
| 관계 데이터 | `scatterplot`, `regplot`, `pairplot` |
| 시계열 데이터 | 선 그래프, 7일 이동 평균, IQR 기반 이상치 탐지 |
| 리샘플링 | 다운샘플링(일 단위 평균), 업샘플링(선형 보간), min/max 집계 |
| 이동평균 | SMA(단순 이동평균), EMA(지수 이동평균), 변동성 탐색 |
| 금융 데이터 | 이동평균 계산, 주간 단위 리샘플링·변동성 계산 |
| 정규분포 | PDF, CDF, PPF 계산 |
| 기술통계 | 평균·중앙값 비교, IQR 이상값 제거, 왜도·첨도 분석 |
| 가설검정 | t-검정, 카이제곱 검정, ANOVA |
| 통계적 시각화 | 박스플롯, KDE + 히스토그램, 카이제곱 클릭률 분석 시각화 |
