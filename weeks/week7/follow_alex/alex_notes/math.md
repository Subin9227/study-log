# int

int는 정수(Integer)를 나타내는 파이썬의 기본 자료형으로, 소수점이 없는 양수, 음수, 그리고 0을 표현합니다

## sum
sum은 리스트나 튜플처럼 여러 개의 숫자가 담긴 반복 가능한(Iterable) 객체를 입력받아, 그 안의 모든 요소의 합을 계산하여 반환하는 파이썬 내장 함수입니다.

## for
for는 정해진 범위나 리스트 등의 요소를 순서대로 하나씩 꺼내어 반복 작업을 수행할 때 사용하는 제어문입니다. 반복 횟수가 명확하거나 순회할 대상이 있을 때 주로 사용됩니다.

## while
while은 지정한 조건식이 참(True)인 동안 특정 코드 블록을 계속해서 반복하여 실행하는 제어문입니다. 반복 횟수가 정해지지 않고, 특정 조건이 만족될 때까지 루프를 돌려야 할 때 주로 사용됩니다.
