## 개선사항
1. 새벽까지 공부한 시간일 경우, 전 날 공부로 들어가도록 수정 (06-21)
2. 과제(weeks)와 공부기록(logs) 폴더 분리 + 일자별 공부시간 달력 추가 (06-25)
3. 수동 실행 시 날짜를 직접 지정할 수 있도록 workflow_dispatch 입력 필드 추가 (06-28)
   - `auto`: 실행 시각 기준 자동 판단 (18시 이후 → 오늘, 이전 → 어제)
   - `today` / `yesterday`: 오늘 또는 어제 강제 지정
   - `manual`: 날짜 직접 입력 (YYYY-MM-DD)
4. 달력에 주차별(월~일) 합계 열 추가 + 월별 접기/펼치기 토글, 5월 백필 모드(`backfill`) 추가 (06-30)
   - 주간 합계는 전체 공부시간과 부트캠프 시간을 함께 표시
   - 날짜칸 시간은 06-25부터 표시(이전은 합계에만 포함)
